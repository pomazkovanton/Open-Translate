# Open Translate

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Platform](https://img.shields.io/badge/Platform-Chrome%20%7C%20Firefox-blue.svg)](#)
[![Manifest Version](https://img.shields.io/badge/Manifest-V3-brightgreen.svg)](#)

**Open Translate** — это универсальное, минималистичное и высокопроизводительное кроссбраузерное расширение для перевода веб-страниц (работает в Chrome и Firefox). Оно предоставляет пользователю полную свободу выбора ИИ-провайдеров и моделей: от ведущих облачных решений (Gemini, Groq, SambaNova, SiliconFlow, OpenRouter) до полностью локального запуска через Ollama.

Интерфейс расширения выполнен в стильном монохромном дизайне (Zinc dark theme) по стандартам современных премиальных веб-интерфейсов, обеспечивая удобное управление настройками перевода в один клик.

---

## ✨ Основные возможности

*   **⚡ Кроссбраузерность:** Полная поддержка Google Chrome (и любых Chromium-браузеров) и Mozilla Firefox.
*   **🔌 Свобода выбора ИИ:** Поддержка API-ключей Google AI Studio (Gemini), Groq Cloud, SiliconFlow, SambaNova Cloud, OpenRouter, а также локального запуска через Ollama.
*   **🔑 Полная конфиденциальность:** Ваши API-ключи хранятся исключительно локально в браузере (`chrome.storage.local`) и передаются напрямую на сервера провайдеров.
*   **🔄 Динамический список моделей:** Автоматическое получение актуальных и доступных моделей из API Groq, SiliconFlow, SambaNova Cloud и OpenRouter.
*   **🖥️ Локальный запуск через Ollama:** Автоматическое сканирование локального сервера и предоставление списка установленных на вашем ПК моделей.
*   **🎨 Два продвинутых режима перевода:**
    *   **Замена:** Полная замена оригинального текста на странице переведенным.
    *   **Рядом:** Отображение перевода рядом с оригиналом в скобках. Стиль перевода кастомизируется (настраиваемый цвет и линия подчеркивания) и адаптирован так, чтобы не ломать верстку сайтов.
*   **⌨️ Горячие клавиши:**
    *   `Ctrl+Shift+Y` (или `Cmd+Shift+Y` на macOS) — перевести страницу.
    *   `Ctrl+Shift+U` (или `Cmd+Shift+U` на macOS) — восстановить оригинальный текст.
*   **🛡️ Оптимизация запросов:** Умная сегментация контента и пакетная отправка запросов для предотвращения таймаутов сервис-воркеров и обхода лимитов API.

## 🛠️ Установка

### 🌐 Google Chrome (и Chromium-браузеры: Edge, Opera, Yandex)
1. Скачайте репозиторий в виде ZIP-архива и распакуйте его, либо клонируйте с помощью Git:
   ```bash
   git clone https://github.com/ваша-ссылка/open-translate.git
   ```
2. Откройте Chrome и перейдите на страницу расширений: `chrome://extensions/`.
3. Включите **«Режим разработчика»** (Developer mode) в правом верхнем углу.
4. Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked) в левом верхнем углу.
5. Выберите **корневую папку** репозитория (где находится файл `manifest.json` для Chrome).

### 🦊 Mozilla Firefox
Для Firefox подготовлена отдельная сборка, адаптированная под архитектуру неперсистентных фоновых скриптов Firefox MV3:
1. Откройте Firefox и перейдите по адресу: `about:debugging#/runtime/this-firefox`.
2. Нажмите кнопку **«Загрузить временное дополнение...»** (Load Temporary Add-on...).
3. Выберите файл `manifest.json` из папки **`firefox/`** в данном репозитории.

## 🚀 Настройка провайдеров

### 1. Бесплатные облачные провайдеры
*   **Google AI Studio (Gemini):** Зарегистрируйтесь на [Google AI Studio](https://aistudio.google.com/), создайте бесплатный API-ключ. Рекомендуется использовать быструю и эффективную модель `gemini-2.5-flash`.
*   **Groq Cloud:** Создайте бесплатный ключ в [Groq Console](https://console.groq.com/keys). Перевод на моделях семейства Llama 3.1 / 3.3 происходит практически мгновенно.
*   **SambaNova Cloud:** Получите ключ на [SambaNova Cloud](https://cloud.sambanova.ai/). Доступны мощные бесплатные модели Llama 3.1 70B и 405B.
*   **SiliconFlow:** Зарегистрируйтесь на [siliconflow.com](https://siliconflow.com), сгенерируйте API-ключ и используйте бесплатные модели (например, `Qwen 2.5` или `Llama 3.1`).

### 2. Локальный запуск через Ollama
1. Установите [Ollama](https://ollama.com/) на ваш компьютер.
2. Скачайте нужные модели (например, Qwen2.5 или Llama3.2) через терминал:
   ```bash
   ollama run qwen2.5:3b
   ollama run llama3.2
   ```
3. Для доступа расширения к локальному API Ollama необходимо запустить сервер с поддержкой CORS (`OLLAMA_ORIGINS="*"`).
   
   **Инструкция для Windows (PowerShell от имени Администратора):**
   ```powershell
   stop-service ollama -force -erroraction silentlycontinue
   stop-process -name ollama -force -erroraction silentlycontinue
   $env:OLLAMA_ORIGINS="*"
   ollama serve
   ```
   *(Не закрывайте окно консоли во время работы расширения).*

4. В интерфейсе расширения выберите провайдера **Ollama** — список ваших локальных моделей подтянется автоматически.

## 🔧 Устранение неполадок

*   **Ошибка подключения к Ollama (403 Forbidden):**
    Убедитесь, что Ollama запущена с переменной окружения `OLLAMA_ORIGINS="*"`. Без этой настройки браузер блокирует запросы расширения из соображений безопасности.
*   **Ошибка 400 Bad Request на SambaNova или SiliconFlow:**
    В некоторых регионах или при использовании некоторых тарифов API-провайдеры строго валидируют JSON-схему запросов. Данное расширение автоматически обрабатывает и форматирует ответы без принудительного включения жесткого JSON-режима на сервере, предотвращая подобные ошибки.
*   **Перевод «Рядом» не отображается:**
    Мы добавили принудительные CSS-стили `display: inline !important; opacity: 1 !important; visibility: visible !important;` для блоков перевода. Это гарантирует отображение перевода даже на сайтах со сложной версткой или агрессивными сбросами стилей.
